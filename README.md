# Antarctic Penguin Project

Hi, in this project I am going to train my geospatial skills on analyzing and mapping the [Antarctic Penguin Tracking & Colony Data](https://www.kaggle.com/datasets/ibrahimqasimi/antarctic-penguin-tracking-and-colony-data?select=MAPPPD_Antarctic_Penguin_Colonies.csv) I found on kaggle.

To use it for yourself, the important libraries are in the .txt file, just use

Using Bash

```shell 
pip install requirements.txt
```

or using conda

```shell
conda install requirements.txt
```
